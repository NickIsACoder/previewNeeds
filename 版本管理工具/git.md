### git使用流程

#### 克隆远程版本库
```
git clone https://github.com/NickIsACoder/justForTest.git
```

#### 拉取最新版本
```JavaScript
git pull
```

#### 提交代码
```JavaScript
git add . 
git commit -m '提交修改'
git push
```

#### 分支
``` JavaScript
// 查看分支
git branch

// 创建本地分支

// 提交本地分支

// 切换分支
git checkout test

// 删除本地分支
git branch -d test

// 删除远程分支
git push origin -d test 
```
